# 游戏图标资源

## 说明

这个目录用于存放游戏中使用的图标图片。

## 文件命名规范

图标文件应命名为：`icon-1.png`, `icon-2.png`, `icon-3.png` ... `icon-30.png`

## 图标要求

- **格式**：PNG（支持透明背景）
- **尺寸**：建议 128x128 像素或更高
- **数量**：至少需要 30 个不同的图标
- **风格**：建议使用统一风格的图标，如：
  - 水果系列（苹果、香蕉、橙子、葡萄等）
  - 动物系列（猫、狗、兔子、熊等）
  - 表情系列（笑脸、哭脸、惊讶等）
  - 符号系列（心形、星星、月亮、太阳等）

## 临时方案

在正式图标准备好之前，可以：
1. 使用纯色方块作为占位符
2. 使用文字标识（1, 2, 3...）
3. 从免费图标库下载（如 iconfont, flaticon）

## 推荐资源

- [iconfont](https://www.iconfont.cn/)
- [Flaticon](https://www.flaticon.com/)
- [Icons8](https://icons8.com/)
