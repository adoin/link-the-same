# 微信小程序连连看游戏

一个基于微信小程序原生框架开发的连连看游戏项目。

## 技术栈

- TypeScript
- SASS/SCSS
- 微信小程序原生框架
- Skyline 渲染引擎

## 开发环境要求

- Node.js >= 16
- pnpm >= 8
- 微信开发者工具

## 快速开始

### 1. 安装依赖

```bash
pnpm install
```

### 2. 在微信开发者工具中打开

1. 打开微信开发者工具
2. 导入项目，选择项目根目录
3. 开始开发

## 游戏规则

1. **目标**：消除所有配对的图标
2. **消除条件**：两个相同的图标可以通过不超过3条直线（最多2个转角）连接
3. **计分**：
   - 基础分：每对消除 +10 分
   - 连击加成：连续消除有额外加分
   - 时间加成：剩余时间越多，最终得分越高
4. **游戏结束**：
   - 胜利：所有图标消除完成
   - 失败：时间用尽或无可消除路径
